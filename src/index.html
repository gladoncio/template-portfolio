Caja registradora

Este proyecto consistió en el desarrollo de un sistema de caja para un cliente que gestiona un restaurante y un almacén. La solución está diseñada para optimizar el proceso de ventas, gestionar el inventario y proporcionar un flujo de trabajo eficiente en el punto de venta (POS). El sistema está integrado con una gaveta de dinero y una impresora de recibos para asegurar una experiencia fluida y profesional tanto para los clientes como para el personal del restaurante y almacén.
